Usage Notes

Implementation.

Fonts

Helvetica Now Display — proprietary (Monotype). If not installed, the system font stack provides near-identical rendering on macOS/Windows/Linux. No license required for local report generation.

DM Mono — loaded from Google Fonts via the tokens.css import. Used for all numerical data, labels, and monospace elements.

How to use

Include tokens.css in your HTML report, or inline the :root block. Then reference tokens as var(--doob-teal), etc.

See report-template.html for the ready-to-use autoresearch report template.
